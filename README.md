# Di cosa si tratta? 
Questo repository funge da snapshot open-source per le release maggiori del sito [studiomessini.com](https://studiomessini.com) <br> 
contiene tutto il materiale ridistribuibile del sito, con la sola esclusione di dati soggetti a revisione umana quali le informazioni presenti nella pagina "ho formato". 

## Tecnologie 
Il sito si basa su tecnologie Open Source distribuite in origine sotto licenze permissive (MIT, si veda sotto)
In particolare il sito fa uso di: 
- Astro 
- TailwindCSS
- DaisyUI 
- Preact 
Tali tecnologie sono da considerarsi esterne alla licenza della repository e il copyright dei marchi sopracitati appartiene esclusivamente ai loro detentori e/o team di sviluppo.
